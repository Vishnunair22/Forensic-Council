# Forensic Council Backend

Multi-Agent Forensic Evidence Analysis System - Backend API

## Description

This is the backend API for the Forensic Council application, a multi-agent system for forensic evidence analysis.

## Features

- Multi-agent architecture for evidence analysis
- Support for image, audio, video, and metadata analysis
- LLM-powered investigation workflows
- Evidence chain of custody logging
- Secure authentication and authorization

## Technology Stack

- FastAPI
- Python 3.12
- PostgreSQL
- Redis
- Qdrant (vector search)
