# Data Schemas & Models

This document defines the key Pydantic (Backend) and TypeScript (Frontend) models used across the Forensic Council architecture.

## 1. AgentFinding

Represents a single conclusion drawn by one of the 5 agents.

```json
{
  "finding_id": "uuid",
  "agent_id": "agent-1-image",
  "agent_name": "Image Forensics",
  "finding_type": "splicing_detection",
  "status": "complete",
  "confidence_raw": 0.94,
  "calibrated": true,
  "calibrated_probability": 0.88,
  "court_statement": "The region at (120,400) exhibits quantization matrices statistically distinct from the background...",
  "robustness_caveat": false,
  "robustness_caveat_detail": null,
  "reasoning_summary": "Detected spliced artifact via GaussianMixture."
}
```

*   `confidence_raw`: The raw return from the ML subprocess script (e.g., the IsolationForest score).
*   `calibrated_probability`: The final score (0.0 - 1.0) after the LLM parses the raw baseline against context.

## 2. InvestigationReport

The final synthesized report generated by the Council Arbiter containing all findings, disputes, and the cryptographic signature.

```json
{
  "report_id": "uuid",
  "session_id": "uuid",
  "case_id": "CASE-1234",
  "executive_summary": "The media contains multiple highly-confident anomalies pointing toward synthetic generation and post-capture splicing.",
  "per_agent_findings": {
     "agent-1-image": [ ...AgentFinding... ],
     "agent-2-audio": [ ...AgentFinding... ]
  },
  "cross_modal_confirmed": [ ...AgentFinding... ],
  "contested_findings": [
    {
       "finding_a": {...},
       "finding_b": {...},
       "verdict": "USER_OVERRIDE"
    }
  ],
  "incomplete_findings": [],
  "uncertainty_statement": "Audio deepfake detection was inconclusive due to high background noise.",
  "cryptographic_signature": "HEX_STRING...",
  "report_hash": "SHA256_HASH...",
  "signed_utc": "2026-03-02T14:30:00Z"
}
```

## 3. HITLCheckpoint

When an agent needs human intervention during the ReAct loop, it yields this object over the WebSocket and awaits a REST decision response.

```json
{
  "checkpoint_id": "uuid",
  "session_id": "uuid",
  "agent_id": "agent-3-object",
  "agent_name": "Object Detection",
  "brief_text": "I detected a lighting direction anomaly, but EXIF data says the flash fired. Should I trust the pixel data or the metadata?",
  "decision_needed": "APPROVE, REDIRECT, or TERMINATE",
  "created_at": "2026-03-02T14:31:00Z"
}
```

## 4. EvidenceArtifact

Stored in Postgres to maintain chain of custody on uploaded files before analysis.

```json
{
  "artifact_id": "uuid",
  "case_id": "CASE-1234",
  "investigator_id": "INV-XYZ",
  "original_filename": "suspect_video.mp4",
  "sha256_hash": "e3b0c44298fc1c14...",
  "size_bytes": 14205512,
  "mime_type": "video/mp4",
  "uploaded_at": "2026-03-02T14:28:00Z"
}
```
