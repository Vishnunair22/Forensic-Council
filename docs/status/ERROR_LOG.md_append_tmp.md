| 304 | **UI stuck on "Initializing forensic tools" for 2+ minutes before tasks start** | 🔴 Critical | **RESOLVED** | Root cause: Agents 1 and 3 had heavy ML tools (`extract_evidence_text` using EasyOCR, and `scene_incongruence` using CLIP) embedded in their `build_initial_thought()` methods. Because this method runs *before* the ReAct loop starts (Step 5 of `run_investigation`), the entire agent blocked while these multi-gigabyte models loaded into CPU RAM. Since no tasks were `IN_PROGRESS` yet, the heartbeat correctly reported "Initializing..." without progressing. Fix: Removed these heavy ML pre-screens from `build_initial_thought()`. Agents now instantly build their initial thought using fast tools (file hash, EXIF) and jump immediately into the ReAct loop, allowing the UI to instantly show the first task running. |
